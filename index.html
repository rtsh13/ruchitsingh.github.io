Research

Efficient GPU Virtual Memory Management in Multi-Tenant Serverless LLM Inference Systems

Principal Goal: Improve virtual memory management to address resource contention, fairness, and user isolation challenges in multi-model LLM serving environments.

Designing a new GPU operating system that virtualizes GPU compute and memory, enabling all GPU memory in a machine to be viewed as universal memory accessible from any GPU. The project implements CPU memory virtualization techniques for GPUs, addressing the limitations of existing systems like ServerlessLLM that use run-to-completion strategies leading to unfairness, resource underutilization, and high tail latency.

Supervised by Dr. Yuvraj Patel, University of Edinburgh

A Dynamic Shift in E-Voting: Delineating Architecture and Regulation via Blockchain

Published: Book Chapter in "Recent Trends and Challenges in Law"

Proposes a blockchain-based decentralized e-voting architecture for India's electoral system, addressing security concerns in offline-based electronic voting machines. The research examines constitutional preconditions for adult suffrage and analyzes the legislative framework including eligibility, single vote representation, transparency, privacy, and data integrity in e-voting processes.

Incorporates legal analysis of the Information Technology Act 2000, Indian Penal Code 1860, and Representation of the People Act 1951 to address implementation ramifications, while exploring the governance role of the Election Commission of India and cryptocurrency-specific legislation requirements for blockchain-based elections.
